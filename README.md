# The Step Up

**Tonight's prompts. Step up when you're ready.**

A creative-prompt board for an AA meeting. 25 cards across 5 categories — Music, Poetry & Writing, Visual Art, Dance, Film — plus a featured visual sequence, *John's Recovery Arc: Face it. Trace it. Replace it.*

Live: <https://tiquermedia.github.io/the-step-up>

## Stack

- Vanilla HTML, CSS, JS — no build step, no dependencies
- Source Serif 4 + Inter via Google Fonts
- "Claimed" state persisted in `localStorage` per device

## Run locally

Open `index.html` in any browser. Or:

```sh
python3 -m http.server 8000
# then visit http://localhost:8000
```

## Adding images

See [`images/README.md`](images/README.md) for filenames and placement. The page renders styled placeholders for any missing images.

## Repo layout

```
index.html       # full page
style.css        # design system + components
main.js          # claim/release toggle + reset
images/          # all images live here (see images/README.md)
```
